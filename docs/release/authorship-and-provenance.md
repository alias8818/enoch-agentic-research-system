# Authorship and provenance framing

This project releases software for an autonomous research-control pipeline and a corpus of AI-generated research artifacts.

The maintainer/operator should be credited as the system operator and release maintainer, not as the human author of generated papers or generated scientific claims.

Recommended language:

> The reports produced by this system are AI-generated research artifacts created from automated run notes, evidence bundles, claim ledgers, and reproducibility traces. The maintainer releases the corpus for inspection and critique but does not claim personal authorship of the generated papers, arguments, or prose.
