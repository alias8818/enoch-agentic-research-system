# ArXiv Markdown Draft Prompt

Write a reviewable arXiv-style research paper draft from the supplied evidence bundle and claim ledger.

Rules:
- Use only supplied evidence. Do not invent metrics, baselines, citations, datasets, or hardware details.
- Mark missing literature citations as TODO rather than fabricating references.
- Include limitations and negative/failed findings.
- Phrase claims as bounded observations unless evidence explicitly supports broader language.
- Include artifact paths for reproducibility.

Required sections: Abstract, Introduction, Related Work, Method, Experimental Setup, Results, Discussion, Limitations, Reproducibility Statement, Conclusion, References, Appendix Artifact Manifest.
